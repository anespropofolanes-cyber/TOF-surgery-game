STAGE 1 — PART A
為什麼 TOF 要依年齡選擇手術？
同一個診斷，不同年齡，完全不同的手術決策
核心概念：姑息 vs 矯正
TOF 的手術分兩大類：

完整矯正術Primary / Complete Repair
一次手術同時修補 VSD + 解除 RVOT 阻塞，根本解決問題。

姑息性手術Palliative Surgery
先做一個暫時性的處置，讓病人撐到可以做矯正術的時機。

選哪條路，取決於年齡、體重、肺動脈發育、全身狀況。
關鍵指標：Qp/Qs
Qp（肺血流量）vs Qs（體循環血流量）

正常：Qp/Qs = 1（肺和體循環各跑一遍）
TOF：Qp/Qs < 1（肺血流不足 → 缺氧）

姑息手術的目標就是讓 Qp/Qs 提升到接近 1，讓寶寶可以撐到矯正術。
但 Qp/Qs 太高也不好：肺血流過多 → 心臟容積負荷過重。

STAGE 2 — PART A
新生兒期：BT Shunt 的麻醉
Blalock-Taussig Shunt — 為什麼做？怎麼麻醉？
BT Shunt 是什麼？
在鎖骨下動脈Subclavian artery和肺動脈之間放一條人工血管，製造一條額外的肺血流路徑。

目的：讓更多血液進入肺部換氣，提升 SpO₂，讓寶寶撐到矯正術。

現代多做 Modified BT Shunt（mBTS）：用 Gore-Tex 人工血管連接，可控制管徑大小。
麻醉核心挑戰：Qp/Qs 平衡
BT Shunt 手術中，麻醉師要維持的不是「正常血壓」，而是Qp/Qs 的平衡。

Shunt 太多（Qp/Qs 太高）：肺血流過多 → 體循環「被偷走」 → 低血壓、代謝性酸中毒
Shunt 太少（Qp/Qs 太低）：肺血流不足 → SpO₂ 仍然很低 → 缺氧持續

目標：SpO₂ 維持在 75–85%（不追求正常值）
新生兒期 CPB 的額外風險
新生兒心肺繞道Cardiopulmonary Bypass, CPB風險特別高：

低體重：體外循環管路血液稀釋效應更大，凝血因子耗損更嚴重
腦部發育未完全：低溫和 CPB 對神經系統的長期影響更顯著
器官功能不成熟：腎、肝對低灌流的耐受度差

這是為什麼高風險新生兒優先選姑息手術而非直接矯正的原因。
麻醉注意事項
術側手臂不能量血壓：BT Shunt 做在患側鎖骨下動脈，術中和術後患側手臂血壓會偽性降低
→ 必須在對側手臂或下肢監測血壓，反映真實中央主動脈壓

吸入氧濃度（FiO₂）管理：高 FiO₂ 會降低肺血管阻力 → Qp↑ → 體循環血流被偷走
→ 通常使用 空氣 + 低 FiO₂ 以維持肺-體循環平衡

動脈監測：建議非患側動脈留置，術中連續監測

STAGE 3 — PART A
3–6 個月：選擇性完整矯正術
最佳手術時機，多模式麻醉策略
為什麼 3–6 個月是最佳時機？
技術上比新生兒期容易（體重更大、解剖更清楚）
住院時間更短、不良事件發生率更低
減少需要跨瓣環補片Transannular Patch, TAP的機會
避免 RVH 過度——RVH 越嚴重，術後心室功能恢復越慢

分階段修補（先姑息再矯正）的累積住院時間是直接矯正的近兩倍（32 天 vs 19 天）
多模式麻醉方案
核心策略：限制呼吸抑制劑 → 促進早期拔管

Dexmedetomidine：唯一持續被研究證實可縮短術後呼吸器使用時間的藥物
→ 鎮靜、鎮痛、降低交感張力，不壓制呼吸

區域麻醉技術：
豎脊肌平面阻滯Erector Spinae Plane Block, ESPB
橫胸肌平面阻滯Transversus Thoracis Plane Block, TTPB
→ 有效減少鴉片類藥物使用，降低術後呼吸抑制風險
術後 JET 預防：Esmolol 的角色
交界性異位性心動過速Junctional Ectopic Tachycardia, JET是 TOF 矯正術後最常見的嚴重心律不整。

研究顯示：術後常規使用 Esmolol（短效 β 阻斷劑）可維持血流動力學穩定、減少 JET 發生率。

機制：Esmolol 降低心率 → 充填時間↑ → 心輸出量改善 → 也抑制 RVOT 的自律性增高
術中評估標準
體外循環撤除後，TEE 確認：
RV 壓力應 < 70–80% 主動脈壓（若超過，表示 RVOT 解除不完全）
VSD 修補完整性（無殘存分流）
雙心室功能評估

RV 壓力比 > 70–80% → 需要外科重新評估 RVOT

STAGE 4 — PART A
成人 TOF：二次手術麻醉
肺動脈瓣置換術（PVR）— 最常見的成人再介入手術
為什麼需要 PVR？
TOF 矯正術常需要切開或擴大 RVOT，導致術後肺動脈瓣逆流（PR）。

長期 PR → RV 容積超載 → RV 擴大 → RV 功能下降

PVR 適應症（符合至少 2 項）：
RV 收縮末期容積指數 > 80 mL/m²
RV 舒張末期容積 ≥ 2倍 LV 舒張末期容積
RV EF ≤ 46%
LV EF ≤ 50%
運動耐力進行性下降
TPVR vs SPVR
TPVRTranscatheter PVR：經導管，微創，適合有適當解剖條件者
SPVRSurgical PVR：開胸手術，適用於合併其他需手術修復問題者

術前建議先做電生理檢查：約 25% 病人可發現可處理的心律不整基質（尤其 VT）
→ 若有，在 PVR 同時一併處理，避免術後猝死風險
成人 TOF 麻醉的特殊挑戰
誘導時最危險：已擴大的 RV 對 SVR 驟降耐受度極差

誘導藥物選擇：
Etomidate：對血流動力學影響最小，是成人 ACHD 誘導首選之一
Propofol：謹慎使用，緩慢給藥，避免快速推注

高風險病人（嚴重 RV 擴大、低 RV EF）：
→ 考慮預防性準備周邊 VA-ECMO（體外膜氧合），萬一心臟無法撐過誘導或手術可立即啟動
成人 ACHD 特殊監測考量
動脈導管位置：既往 BT Shunt 的患側手臂血壓偽性低 → 必須對側或股動脈

靜脈通路：許多成人因兒童期多次心導管，股靜脈/動脈可能閉塞 → 術前評估血管解剖

心外膜節律器位置：可能在橫膈下方，腹部手術前需影像確認位置

靜脈輸液過濾器：有殘存分流者必用（矛盾性栓塞預防）

CVP
中心靜脈導管
CVP 變化的臨床意義
▶
決策依據：RV 功能不全和肺高壓的嚴重程度、手術風險、麻醉師解讀能力

▶
